To deploy the project, follow these steps:

1. Clone the repository using git

```bash
git clone https://github.com/QuentiumYT/Instalike.git
cd Instalike/
```

2. Install dependencies

```bash
npm i

yarn
```

3. Copy the .env.example into .env

4. Run the projet

```bash
yarn start
```
